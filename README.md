# PrimeTimeTradeBot

PrimeTimeTradeBot — это бот на базе Telegram, предназначенный для анализа торговых данных. Он обрабатывает файлы `.csv`, полученные с сайта [finam.ru](https://www.finam.ru/quote/moex/gdu4202309/export/), и предоставляет пользователям информацию о лучших часах для торговли на основе их активов.


## **Работа с ботом:**

#### **1. Использование**
**Команды**
- **/start** - Выводит приветственное сообщение и инструкции по использованию бота.
- **/help** - Предоставляет инструкцию по загрузке данных.
- **/menu** - Показывает главное меню с сохраненными активами и доступными действиями.
- **/summary** - Отправляет сводку по всем загруженным активам.

**Загрузка данных**
<br>Загрузите файл .csv с данными по активу через интерфейс Telegram бота.
Убедитесь, что файл соответствует требованиям:
- Формат: .csv
- Периодичность: 1 час
- Имя файла: сокращенное название актива
- Формат времени: чч
- Разделитель полей: точка с запятой ;
- Формат записи: DATE, TIME, OPEN, HIGH, LOW, CLOSE, VOL
- Время: Московское

#### **2. Как это работает**
Бот обрабатывает загруженные данные и сохраняет их в папку пользователя.
Программа анализирует данные, строит графики и сохраняет результаты.
Бот отправляет пользователю результаты анализа в формате текста и изображений.
Реализована функция меню и сохраненых активов для пользователя
